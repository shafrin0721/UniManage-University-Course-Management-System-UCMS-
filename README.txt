UniManage - ASP.NET MVC 5 (.NET Framework) Final Project Scaffold

Target:
- Visual Studio 2017 or higher
- ASP.NET MVC 5
- .NET Framework 4.7.2
- Entity Framework 6
- SQL Server LocalDB or SQL Server Express

How to open:
1. Create/Open an ASP.NET MVC 5 (.NET Framework) solution named UniManage.
2. Replace the generated files with the files in this folder.
3. Restore NuGet packages.
4. Update the connection string in Web.config if needed.
5. Run the app. The database is created automatically through the initializer.
